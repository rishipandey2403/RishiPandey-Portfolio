Overview

This project focuses on analyzing bank loan data to uncover trends, identify high-risk borrowers, and streamline the loan approval process. By leveraging Power BI for interactive data visualization and SQL for data preparation, we can deliver actionable insights that drive informed decisions for financial institutions.

Project Details

Objectives

  • Track total loan applications, funded amounts, and month-over-month performance.
  • Identify and categorize borrowers by risk levels and demographics.
  • Provide an interactive dashboard to monitor loan performance in real-time.

Methodology

  • Data Collection: The financial_loan.csv dataset was used to gather loan information.
  • Data Cleaning & Transformation: SQL scripts (Queries.sql) were executed to filter and preprocess data, ensuring accuracy and consistency.
  • Domain Knowledge: Reference documents (Domain Knowledge Doc.docx and Terminologies in Data.docx) were used to understand financial metrics and industry terminology.
  • Dashboard Creation: Power BI (Bank_Loan_Dashboard.pbix) was utilized for data visualization, creating interactive charts and tables for loan insights.

Key Insights

  • High-risk segments identified through default rates and credit score patterns.
  • Correlation between loan amount, tenure, and customer demographics.
  • Month-over-month trends for loan approvals and disbursements, guiding strategic decisions.
