Overview

The Bank Loan Dashboard project is an interactive solution designed to provide comprehensive insights into bank loan data. It leverages advanced data analytics techniques to assess loan performance, risk factors, and overall financial trends, enabling financial institutions to make data-driven decisions.

Project Details

Objectives

  • Visualize key performance indicators such as total loan applications, funded amounts, and default rates.
  • Identify high-risk borrower segments and monitor trends over time.
  • Provide an interactive dashboard for real-time monitoring and strategic decision-making.

Methodology

Data was collected from the financial_loan.csv dataset and preprocessed using SQL queries. Domain-specific documents (Domain Knowledge Doc.docx and Terminologies in Data.docx) informed the KPI selection. Interactive dashboards were built using both Power BI and Tableau to visualize the data.

Key Insights

  • Identification of high-risk customer segments based on default rates.
  • Trend analysis of loan approvals and disbursements over different time periods.
  • Correlation between loan amounts, tenure, and repayment behaviors.
