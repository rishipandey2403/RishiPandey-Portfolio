Project Overview

The Bank Loan Dashboard project is an interactive Power BI solution designed to provide comprehensive insights into bank loan data. It leverages advanced data analytics techniques to assess loan performance, risk factors, and overall financial trends, enabling financial institutions to make data-driven decisions.

Problem Statement

Banks and financial institutions require a real-time, data-driven approach to understand the dynamics of loan applications and approvals. Traditional reporting methods often fail to provide actionable insights into risk factors, customer segmentation, and performance metrics, leading to sub-optimal decision-making.

Objectives

  • Visualize key performance indicators (KPIs) such as total loan applications, funded amounts, and default rates.
  • Identify high-risk borrower segments and trends over time.
  • Enhance decision-making through interactive, drill-down reports.

Methodology

The project involves multiple stages of data processing and visualization:

  • Data Collection: The primary dataset, financial_loan.csv, was used to gather loan information.
  • Data Preparation: Data was cleaned and transformed using SQL queries (refer to Queries.sql and Queries.docx).
  • Domain Knowledge: Supporting documents (Domain Knowledge Doc.docx and Terminologies in Data.docx) provided insights into key financial metrics and loan dynamics.
  • Visualization: The prepared data was visualized in Power BI (using the Bank_Loan_Dashboard.pbix file) to create interactive dashboards and reports.

Key Insights

  • Identification of high-risk customer segments through default analysis.
  • Trends in loan applications and approval rates over different time periods.
  • Correlation between loan amounts, customer demographics, and repayment behavior.
